43class GreenSSLSocket(_original_sslsocket): 

44 """ This is a green version of the SSLSocket class from the ssl module added 

45 in 2.6. For documentation on it, please see the Python standard 

46 documentation. 

47 

48 Python nonblocking ssl objects don't give errors when the other end 

49 of the socket is closed (they do notice when the other end is shutdown, 

50 though). Any write/read operations will simply hang if the socket is 

51 closed from the other end. There is no obvious fix for this problem; 

52 it appears to be a limitation of Python's ssl object implementation. 

53 A workaround is to set a reasonable timeout on the socket using 

54 settimeout(), and to close/reopen the connection when a timeout 

55 occurs at an unexpected juncture in the code. 

56 """
