100def serve(sock, handle, concurrency=1000): 

101 """Runs a server on the supplied socket. Calls the function *handle* in a 

102 separate greenthread for every incoming client connection. *handle* takes 

103 two arguments: the client socket object, and the client address:: 

104 

105 def myhandle(client_sock, client_addr): 

106 print("client connected", client_addr) 

107 

108 eventlet.serve(eventlet.listen(('127.0.0.1', 9999)), myhandle) 

109 

110 Returning from *handle* closes the client socket. 

111 

112 :func:`serve` blocks the calling greenthread; it won't return until 

113 the server completes. If you desire an immediate return, 

114 spawn a new greenthread for :func:`serve`. 

115 

116 Any uncaught exceptions raised in *handle* are raised as exceptions 

117 from :func:`serve`, terminating the server, so be sure to be aware of the 

118 exceptions your application can raise. The return value of *handle* is 

119 ignored. 

120 

121 Raise a :class:`~eventlet.StopServe` exception to gracefully terminate the 

122 server -- that's the only way to get the server() function to return rather 

123 than raise. 

124 

125 The value in *concurrency* controls the maximum number of 

126 greenthreads that will be open at any time handling requests. When 

127 the server hits the concurrency limit, it stops accepting new 

128 connections until the existing ones complete. 

129 """ 

130 pool = greenpool.GreenPool(concurrency) 

131 server_gt = greenthread.getcurrent() 

132 

133 while True: 

134 try: 

135 conn, addr = sock.accept() 

136 gt = pool.spawn(handle, conn, addr) 

137 gt.link(_stop_checker, server_gt, conn) 

138 conn, addr, gt = None, None, None 

139 except StopServe: 

140 return
