153 

154 The *evtype* argument is either the constant READ or WRITE. 

155

160

162 that the file was closed. 

163

165 prepare a Python object as being closed, pre-empting further 

166 close operations from accidentally shutting down the wrong OS thread.

190 Any current listeners must be defanged, and notifications to
